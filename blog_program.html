Step 5 / Making Pallette

Program Pallette

You will need:

  • Tested electronic circuit
  • Firmware source files
  • Programming cable
  • CC Debugger
  • Bluegiga update tool
  • Windows environment

The firmware runs on the Bluetooth enabled BLE113 microcontroller. The files are on this Github repo.

To program the microcontroller, you will need this cable.

and CC Debugger

You will need to install the CC Debugger driver, and the Bluegiga SDK on the Windows environment.

To program the microcontroller, we need to assemble the cable and CC Debugger, and upload the firmware file to the microcontroller using the Bluegiga update tool. The video below shows how.

video

To test whether the firwmare is running as expected, we need to install the app and see whether the app is showing correct values.

The Android app is mainly for debugging purposes. The files are on this Github repo.
