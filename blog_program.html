Step 5 / Making Pallette

Program Pallette

You will need:

  • Pallette sensor band
  • Firmware source files
  • Programming cable
  • CC Debugger
  • Bluegiga update tool
  • Windows environment
  • Pallette debugging app
  • Android device

The firmware runs on the Bluetooth enabled BLE113 microcontroller. We have made two versions of firmware: the debugging firmware is for checking whether the sensor band is working correctly; the mouse controller firmware enables the sensor band to control a mouse cursor using your tongue. First, you will need to install the CC Debugger driver, and the Bluegiga SDK on a Windows environment.

Program the debugging firmware

You will need to download the debugging firmware files and install the Pallette debugging app on an Android device. The app will display sensor reading values, so you can see whether they are correct or not.

(how to install the app)

Next, we need to assemble the sensor band with the programming cable and the CC Debugger, and connect the CC Debugger and your Windows environment with a mini USB cable. Upload the debugging firmware file to the microcontroller using the Bluegiga update tool.

Program the mouse controller firmware

Follow the same steps above to program the mouse controller firmware, the firmware files are here.

If everything works correctly, now your sensor band should be able to move a mouse cursor on a computer or a mobile device according to your finger movement. The next step is to seal the sensor band, so it's safe to use it inside your mouth.
